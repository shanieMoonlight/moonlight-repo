üé® The Modern Angular Theming Challenge

Building beautiful, accessible Angular applications requires sophisticated theming capabilities that go beyond basic CSS. Modern applications need dynamic theme switching, dark mode support, user customization, and seamless Material Design 3 integration. The @spider-baby/theming package provides a comprehensive solution for all these needs and more.

The Essential Requirements:

  • Dynamic Theme Switching - Change themes at runtime without page reloads
  • Material Design 3 Integration - Full M3 color system and design tokens
  • Dark Mode Support - Seamless light/dark mode with system preference detection
  • Custom Theme Creation - Allow users to create and save personalized themes
  • Performance Optimization - Efficient color generation and DOM updates
  • Hierarchical Theming - Different themes for different sections of your app
  • Smooth Transitions - Visually appealing theme change animations
  • Persistent Preferences - Save user choices between sessions

üìö API Reference

ThemeService

  • currentTheme() - Get current theme
  • setTheme(theme) - Apply theme
  • toggleDarkMode() - Toggle dark mode
  • addCustomTheme(theme) - Add custom theme
  • availableThemes() - Get all themes

ThemeGeneratorService

  • applyTheme(theme, element?) - Apply to element
  • exportThemeAsScss(theme) - Generate SCSS

DynamicThemeConfigService

  • setSystemThemes(themes) - Update themes
  • addSystemTheme(theme) - Add theme
  • resetSystemThemesToInitial() - Reset

ThemeTransitionService

  • transitionWithOverlay() - Overlay transition
  • transitionWithMorph() - Morph transition
