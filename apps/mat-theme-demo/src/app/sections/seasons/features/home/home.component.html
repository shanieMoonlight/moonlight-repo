Seasonal Themes

Explore different themes inspired by the seasons. Each card below uses the ThemeGeneratorService to apply its specific theme locally, without affecting the rest of the application or the other cards. This is done using: 

    
this.themeGenerator.applyTheme(
  this._data().theme,
  `season-card-$}this._data().theme.value{`, // Optional unique class
  this.elementRef.nativeElement // Apply theme just to this component only
);

Clicking "Go to [Season]" will navigate to a dedicated page for that season and apply the theme globally using the ThemeService. This affects elements like the main navbar and demonstrates full-page theme transitions. You can set the global theme like this: 

    
// Using the theme value:
this.themeService.setThemeByValue('autumn');

// Or using the theme object:
this.themeService.setTheme(AUTUMN_THEME);
